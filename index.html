Change the default search engine in Microsoft Edge

Applies to Windows 10

To do this, install a new search engine and then set it as your default. Here's how:

  1. Go to the search engine's website.
  2. Select More actions (...) > Settings > View advanced settings. In the list under Search in the address bar with, choose Add new.
  3. Select the website and then select Add.
    Note: Only search providers that support the OpenSearch standard will appear in this list. If they don't support it yet, here's how they can. Keep checking back to see who's been added to the list.
  4. To set this site as your default search provider, go back to the list under Search the address bar with and choose it.
